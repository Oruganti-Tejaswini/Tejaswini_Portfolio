First Pass Project Details ( In progress )

This is a health care project which mainly focus on the chest X-ray and the diseases that can predict by analyzing the chest x-rays. The data that we gather is from NIH website which consists of alomst 110,000 samples with almost 30,000 unique patient details.

The diseases that predicting by analyzing the chest x-ray are as follows:

  • Atelectasis
  • Pneumothorax
  • Pneumonia Cola
  • Pleural_Thickening
  • Nodule
  • Mass
  • Infiltration
  • Hernia
  • Fibrosis
  • Emphysema
  • Effusion
  • Edema
  • Consolidation
  • Cardiomegaly
  • No Findings
The predicted result should be either one or more diseases.


    Project Description:-

  • Extract the data from the zip and create the new dataset applying data augmentation techniques [Zooming, Rotation and shrinking, brightness] to all the diseases of 10% of the original data.
  • Made the train and test sample split from the bulk data.
  • Implement the ML models [Resnet50, Densenet] to find which algorithm is best in predicting the real world problem.
  • Implement and create the data base in order to get to store the patient and hospital details etc.
  • Deploy this model in AWS.
  • This app helps the patients to get to know their results in their finger tips.
